# 通过chatgtp制作多媒体制作软件

#### 介绍

通过和chatgtp聊天，制作一款多媒体制作软件，包括视频，图片轮播，可以设置多种风格的声音， 调节语音速度/文字字幕速度 ， chatgtp文字和语音交互聊天.

涉及技术：python ，tkinter ， chatgtp，openAI， TTS。
